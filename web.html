Web Developer

Job Description

The engineer will be working in the below aspects: - Systems implementation - System Troubleshooting and monitoring - Deploying new product releases, changes, fixes and enhancements. Requirements: - JAVA (servlet, EJB, WebService) under WebSphere - General web development: JavaScript, jQuery, GWT, Apache Tomcat - Good knowledge of JAVA EE / JAVE SE - Good knowledge of Struts 2 framework - Knowledge of IReport 3.0 - Knowledge of Databases and SQL (DB2, MSSQL,Oracle) - Good knowledge in Unix commands and scripting - Good knowledge of remote tools and shell commands like WinSCP and PuTTY - Medium competence on: AIX, Apache (HTTP server), WebSphere) - Deep Knowledge of Services Protocols (SOAP, REST, JMS) - Deep Knowledge of Message Formats (XML, JSON, MQ) - Deep Knowledge of IBM MQ Queues.
